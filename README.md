# blueprint gamequiz
🌟 FITUR UTAMA APLIKASI
Mode Belajar Dulu: Pengguna bisa membaca soal, jawaban benar (hijau), dan penjelasan di kartu geser sebelum kuis dimulai.
Mode Main Kuis: Game tanya jawab dengan limit 3 nyawa, bonus poin jika menjawab benar beruntun (streak), dan soal yang diacak.
Pengacak Pilihan Jawaban: Opsi jawaban diacak otomatis agar kunci jawaban tidak selalu menjadi pilihan teratas.
Selebrasi Meriah: Ledakan kertas warna-warni (confetti) dan piala emas memantul saat kuis berhasil dimenangkan.
Simpan Skor Tertinggi: Skor paling tinggi disimpan otomatis dan bisa dilihat kapan saja.
Pengaturan Suara & Getar: Opsi menyalakan/mematikan audio game dan efek getar getaran ponsel.
Desain Anti-Overload: Tampilan dinamis yang dapat digulir (scroll) sehingga tidak terjadi error garis kuning-hitam di layar ponsel kecil.
